# Contributor Covenant Code of Conduct

## 私たちの誓い

私たちメンバー、コントリビューター、リーダーは、年齢、体格、目に見える／見えない障害、民族性、性特性、性自認・性表現、経験の度合い、教育、社会経済的地位、国籍、個人的見た目、人種、カースト、色、宗教、性自認や性的指向に関わらず、誰にとっても嫌がらせのないコミュニティ体験を提供することを誓います。

オープンで歓迎的・多様で包括的、健全なコミュニティに貢献する方法で行動し、交流することを誓います。

## 基準

前向きな環境に貢献する行動の例:

- 他人に共感と親切を示す
- 異なる意見・視点・経験を尊重する
- 建設的なフィードバックを行い、優雅に受け入れる
- 過ちによって影響を受けた人に責任を持ち、謝罪し、経験から学ぶ
- 個人ではなくコミュニティ全体にとって何が最善かに焦点を当てる

受け入れがたい行動の例:

- 性的な言葉や画像の使用、あらゆる種類の性的注目や誘い
- トロール行為、侮辱的／軽蔑的なコメント、個人攻撃や政治的攻撃
- 公的または私的な嫌がらせ
- 明示的な許可なく他人の個人情報（物理的・電子メールアドレスなど）を公開する
- その他、職業的な場で合理的に不適切と判断される行為

## 執行責任

コミュニティリーダーは、受け入れがたい行動に対して適切かつ公正な是正措置を取る責任を負います。

## 適用範囲

この行動規範は、コミュニティのすべての空間に適用され、また個人が公の場でコミュニティを代表するときにも適用されます。

## 執行

虐待的、嫌がらせ、その他受け入れがたい行動の事例は、コミュニティリーダーに連絡することで報告できます:

**sasagawa@kent-and-co.com**

すべての苦情は迅速かつ公正にレビュー・調査されます。
